# 🌐 Tentwenty Landing Page Project

This project is a simple **landing page** built with **HTML, CSS (via lbnsss-cdn), and JavaScript**.
It includes two main sections and a sliding image **carousel** for a modern, interactive feel.

---

## 📂 Project Structure

```
.
├── index.html   # Main landing page with sections and carousel
├── script.js    # JavaScript for carousel functionality and interactivity
└── README.md    # Project documentation
```

---

## ✨ Features

* **Landing Page Sections**

  * A clean, structured layout with two different sections.
* **Image Carousel**

  * Sliding image carousel implemented with JavaScript.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Open with Live Server

This project is designed to run locally using **Live Server** (VS Code extension).

Steps:

1. Install the **Live Server** extension in VS Code.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.
4. Your landing page will open in the browser.
